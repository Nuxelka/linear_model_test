# Linear Model Test

Проект посвящён обучению линейной модели для задачи регрессии — предсказания стоимости домов `SalePrice`.

В работе выполнены основные этапы подготовки и анализа данных:

- выбор числовых признаков;
- заполнение пропущенных значений;
- анализ корреляций признаков с целевой переменной;
- визуализация зависимостей между признаками и ценой;
- обучение Ridge-регрессии;
- оценка качества модели с помощью RMSE;
- сравнение модели с baseline-предсказанием по среднему значению;
- стандартизация признаков через `StandardScaler`;
- подбор коэффициента регуляризации `alpha` с помощью `GridSearchCV`;
- объединение обработки данных и модели в `Pipeline`.

Цель проекта — разобраться, как работает линейная регрессия с регуляризацией и как подготовка признаков влияет на качество модели.
